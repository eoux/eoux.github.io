Timeline: Dec 2017 - Nov. 2018

Udacity introduced me into a developer’s world. The experience helped broaden my mind with important concepts I wouldn’t have known before. I welcome you to take a look at what projects I accomplished at the Grow with Google program.

Project 2 | Memory Game

Build on a static starter code, this memory game consists of 16 cards arranged in a grid. The cards are arranged randomly on the grid with symbols faced down. Gameplay rules are very simple: flip over two hidden cards at a time to locate the ones that match! Things I learned and implemented: Javascript functionality & Development Strategy.

Project 7 | Farmers Map

The Farmer’s Market app was the last project I completed with the program. It displays 5 markers showing available markets in the L.A. area. It also includes a search functionality that filters markets by query. Things I learned and implemented: Create-React-App, Google Maps API, and FourSquare API.

Project 5 | My Reads

The My Reads app was a project for me to understand the basic structure and operation of a React-based app. I learned and implemented: Create-React-App Documentation, React Router Documentation, and React API.

Project 6 | Restaurant Reviews

The starter code for this project had a lot of issues. I had to make sure the people using the app were able to have a great mobile and desktop experience while searching for restaurants. Things I learned and implemented: Responsive Design, Service Worker for Offline use, and Accessibility.
